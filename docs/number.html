Numbers (Class)

Simplified usage of certain functions of int and double.

Length (class)

To get length of int or double call length class and then call getNumberLength() or getDoubleLength().


Usage for getNumberLength():

#include "ngapi.cpp"

using namespace ngapi;


int main() {

int num = 123456;// Length is 6

int length = number.length.getNumberLength(num);

console.log(text.toString(length));

success();

}
